🛠 Proyecto: Estación Solar con ESP32

Este proyecto mide el índice UV y la irradiancia solar usando un ESP32 y sensores conectados.

🔧 Componentes

  • ESP32 DevKit
  • Sensor UV analógico (conectado al pin 34)
  • Sensor de luz Adafruit TSL2591
  • Pantalla LCD I2C (16x2)

📡 Funcionalidad

  • Conexión automática a WiFi con portal cautivo (WiFiManager)
  • Lectura y cálculo de índice UV e irradiancia
  • Publicación de datos en tiempo real en Adafruit IO
  • Servidor web integrado en el ESP32 para mostrar datos localmente
